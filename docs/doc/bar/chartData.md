# chartData 数据

> 默认 `[]`     类型 `Array`

图表需要展示的数据集合

示例:
``` js
chartData:[{
    key:3,
    age:100,
    val:21
},{
    key:0,
    age:100,
    val:3
},{
    key:null,
    age:100,
    val:56
},{
    key:5,
    age:100,
    val:34
},{
    key:8,
    age:100,
    val:-54
}]
```

::: tip 提示
底部轴(X轴)显示的内容也需要放在`chartData`数组集合中.
:::
